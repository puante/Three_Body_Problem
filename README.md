# 삼체문제(三體問題) 시뮬레이터 / Three Body Problem Simulator
## 🤔/ 프로젝트 개요
삼체문제를 통해 카오스 이론을 탐구해 보자는 시도에서 출발하였습니다.
본 깃허브는 그 때 사용한 시뮬레이션 값들 / 정보 / 프로그램들을 담아두고 있습니다.

## 사용 방법
1. Build 폴더 또는 릴리스 파일을 확인하고 다운로드한다.
2. 내부에 있는 TBP_simulator.exe를 실행한다.
3. 프로그램을 연 후, 내부의 파라미터에 원하는 값을 넣는다.
4. 이후 저장, 로드, 시뮬레이션의 기능들이 가능하다.

 

### [ 설정에 대한 설명들 ]

질량: 말 그대로 행성의 질량 (kg)

좌표: x와 y로 있으며 좌표를 통해 나타내는 행성의 초기위치 (km)

속도: x성분과 y성분이 나뉜 행성의 초기속도 (m/s)

dt: 각 프레임마다 지나는 시간 (s)
 / dt가 3600이라면 한 프레임 당 한 시간이 지난 후를 의미

총 프레임 수: 보여줄 프레임의 분량

애니메이션 속도: 프레임 사이 대기 시간(ms), 자연수로 설정 가능 

## 주의 사항
- 어떤 작업을 하고 있는 지 아는 게 아니라면 TBP 폴더에 있는 것을 만지지 않는 것을 추천드립니다.
- TBP 폴더 안에 있는 TBP_screen_show 파일은 원래 실행되는 파일이 아닙니다! 그렇다고 해서 삭제하시면 안 됩니다.
메인 파일에서 체계화를 위해 넣어둔 시뮬레이션을 보여주는 파일이기에,
잘못 만지셨다간 프로그램이 완전히 작동하지 않는 오류가 발생할 수 있습니다.
- 애니메이션 속도를 과도하게 내렸다가 컴퓨터 성능이 버티지 못한다면 컴퓨터의 생명이 위험해질 가능성이 존재하니 주의하여 조정해 주세요.
